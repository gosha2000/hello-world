# Markdown Practice

This file contains all essential Markdown syntax.

---

## 1. Emphasis (Жирный / Курсив)

- *Italic text example*
- **Bold text example**
- _Также курсив_
- __Также жирный__
- _Можно **смешивать**_

---

## 2. Lists (Списки)

### Unordered list:
- Item 1
- Item 2
  - Subitem 2a
  - Subitem 2b

### Ordered list:
1. First item
2. Second item
3. Third item
   1. Nested item 3a
   2. Nested item 3b
